# Docker Gen File

This module allows you to create Docker and Docker Compose files for your project.

```
npx docker-gen-file
```

**NOTE**

- Execute this command at root level of your project

---

### Currently Supported Languages & Frameworks

1. Static Builds
   - ReactJs
   - VueJs
   - AngularJs
2. NextJs
3. NodeJs
4. Python
   - Django
5. Php
   - Laravel
